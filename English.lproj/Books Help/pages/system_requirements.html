System Requirements

You need the following in order to run Books

  • A computer that supports MacOS X.
  • Mac OS X 10.4 (or higher)
  • A MacOS X-compatible camera (optional, used for scanning barcodes)
  • An Internet connection (optional, used for retrieving book information online)

While Books can be installed on any disk attached to your computer, we recommend that you install it on a non-removable disk.

Unfortunately, Books 3 is only available on MacOS X 10.4 (and higher). If you use an older version of MacOS X, you may wish to try some of the prior versions:

  • MacOS X 10.3: Books 2.3.3 (beta) or Books 2.2.2 (stable)
  • Versions prior to MacOS X 10.3: Books 1.2.3
  • Note: Books is a MacOS X application, thus there are no versions for MacOS 9 and before.
