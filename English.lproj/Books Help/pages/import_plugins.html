Importerweiterungen

Haben Sie vorher eine andere Anwendung genutzt, um Ihre Büchersammlung zu verwalten, so besteht die Möglichkeit, diese Einträge an Books weiterzureichen - insofern eine entsprechende Importerweiterung angeboten wird.


Folgende Importerweiterungen stehen Ihnen derzeit zur Verfügung:

  • Amazon Importer - Diese Erweiterung importiert Daten einer Amazon-Internetsuche
  • BibTeX Importer - Daten, die mithilfe von Bookpedia erstellt wurden, können an Books weitergereicht werden
  • Booxter Importer - Booxter - Daten können mit dieser Erweiterung in Books integriert werden
  • Collectorz.com Importer - Daten können von Ihren Collectorz.com - Einträgen importiert werden
  • Delicious Library Importer - Importiert Daten, die mit Delicious Library erstellt worden sind
  • Library of Congress Importer - Die Erweiterung importiert Daten einer Library of Congress - Internetsuche
  • LibraryThing Importer - Mit dieser Erweiterung können Sie die Einträge Ihres LibraryThing - Kontos in Books integrieren. Damit diese Daten von der Erweiterung gelesen werden können, besuchen Sie die LibraryThing Extras - Internetseite und exportieren Sie dort Ihre Daten als Textdatei mit Trennzeichen (tab-delimited)
  • Tab-Delimited Text Importer - Daten einer Textdatei mit Trennzeichen (tab-delimited) werden ausgelesen und in Books integriert
  • UIEE Importer - Nutzen Sie diese Erweiterung, um UIEE -Dateien zu importieren, die mit Readerware exportiert worden sind
