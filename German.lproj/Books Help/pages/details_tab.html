Der Reiter „Details”

Das Einstellungsfenster bietet Ihnen viele Möglichkeiten, Books an Ihre persönlichen Bedürfnisse anzupassen. Dazu stehen Ihnen vier verschiedene Reiter zur Verfügung. Einstellungen, die Auswirkungen auf die Zusammenfassung haben, werden innerhalb des Reiters „Details” vorgenommen.


Einstellhinweise

Jeder Anzeigestil bringt eine Auswahl an Feldern mit, die im Fenster Zusammenfassung angezeigt werden. Wenn Ihnen die Auswahl, die für den jeweiligen Anzeigestil getroffen wurde, ausreicht, setzen Sie einfach ein Häkchen vor den Eintrag „Standard-Felder der Erweiterung anzeigen”. Möchten Sie allerdings andere Felder angezeigt bekommen, so entfernen Sie das Häkchen vor diesem Eintrag und geben in der unteren Anzeige die Felder an, die Sie benötigen. Beachten Sie bitte, daß Sie jeweils nur ein Feld pro Zeile angeben sollten. Um eine weitere Zeile anzugeben, bestätigen Sie das Ende einer Zeile mit der Eingabetaste.
