Systemvoraussetzungen

Folgende Zutaten benötigen Sie, um mit Books arbeiten zu können:

  • Einen Computer, der Mac OS X unterstützt.
  • Mac OS X 10.4 oder höher, daß auf diesem Rechner installiert sein muß
  • Books ist es völlig egal, ob dieser Rechner mit einem PowerPC- oder einem Intel-Prozessor ausgestattet ist. Die Anwendung liegt als Universal Binary vor.
  • Eine mit Mac OS X kompatible Kamera (optional - wird für die Strichcode-Erkennung benötigt)
  • Eine Internetverbindung (optional - wird für das „Automatische Ausfüllen” benötigt)

Wenn Sie alle Zutaten kräftig miteinander vermischen, können Sie Books in einem beliebigen Ordner auf Ihrer Festplatte installieren. In der Regel sollte Books in den Programme-Ordner Ihrer Festplatte kopiert werden, aber auch andere Speicherplätze sind möglich.

Leider werden durch Books 3 keine Versionen von Mac OS X vor 10.4 mehr unterstützt. Für Nutzer einer älteren Version von Mac OS X stehen folgende Versionen zur Verfügung:

  • Mac OS X 10.3 oder höher: Books 2.3.3 (Beta-Version) oder Books 2.2.2 (stabile Version)
  • Alle Mac OS X-Versionen vor Mac OS X 10.3: Books 1.2.3
  • Anmerkung: Books wird nicht für das klassische Mac OS angeboten. Alle Versionen vor Mac OS X werden nicht unterstützt.
